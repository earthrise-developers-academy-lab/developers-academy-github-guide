# Getting help

Ask project-specific questions in the issue where the work is tracked.

```markdown
### Help requested

**What I am trying to do**
-

**Where I am blocked**
-

**What I expected**
-

**What happened**
-

**What I already tried**
-

**Relevant screenshot, output, or link**
-
```

Contact a repository steward for access problems, branch recovery, suspected sensitive information, merge conflicts, unexpected files, or permission failures.

Use the inherited organization issue forms to report inaccurate or confusing guide instructions.
